
# Website for managing and summarizing point that students receive from school cooperative store

i made this website for NSC 2024 to solve a problem in cooperative store points system. Currently ,cooperative store points system uses a coupons system to accumulate point ,which is hard keep coupons and the most students lazy to get their coupons when they buy things from the cooperative, which can give a 14 percent dividends

For the reasons stated above, I create this website for managing and summarizing point in digital format for easier and more performance to managing and save the time for manager.

## Features

- Login 
	- for students and cashier or cooperative manager
- Point accumulation system
	- when students buy items and pay at the cashier ,Cashier will ask for student ID and enter total price of goods and student ID in to website and submit
- Check student point 
	- student can login and check their point and dividends they will receive at the end of the academic
	- they also can donate their points to school charity project
	- Data able to reset at the end of the academic
